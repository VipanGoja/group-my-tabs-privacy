Privacy Policy for Group My Tabs

Last updated: April 15, 2026

Group My Tabs is a Chrome extension that helps users organize related tabs in the current Chrome window into Chrome tab groups.

What data the extension accesses

The extension accesses tab information in the current Chrome window, including:

  • tab URLs
  • tab titles
  • tab group state needed to create and manage Chrome tab groups

How the extension uses this data

The extension uses this information only to:

  • analyze tabs in the current Chrome window
  • group related tabs into Chrome tab groups
  • let users rename groups
  • let users manually adjust grouping behavior

What data the extension stores

The extension stores the following locally in Chrome storage on the user’s browser:

  • custom group names
  • manual grouping preferences and overrides

Data sharing

Group My Tabs does not sell, transmit, or share tab data, browsing data, or stored user data with the developer or with third parties.

Remote servers

Group My Tabs does not send tab URLs, titles, browsing activity, or stored extension data to remote servers.

Data retention

Data stored by the extension remains in the user’s local Chrome storage until the user clears extension data or removes the extension.
