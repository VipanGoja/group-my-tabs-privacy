Privacy Policy for Unmessy Tabs

Last updated: May 11, 2026

Unmessy Tabs is a Chrome extension that helps users organize, find, and manage open browser tabs. It provides a side panel for viewing open tabs, grouping related tabs, managing Chrome tab groups, and remembering useful workspace/group state locally in the browser.

Summary

Unmessy Tabs is designed to process tab and group information locally in your browser. The extension does not sell your data and does not send tab URLs, tab titles, browsing history, page contents, group names, or locally stored workspace data to the developer or any third party.

The extension uses Google Analytics 4 to collect limited product usage analytics, such as which extension actions are used. These analytics events are used only to understand and improve the extension. No tab URLs, tab titles, browsing history entries, page contents, search query text, or group names are collected in analytics or sent to Google Analytics.

Data the extension accesses

To provide its tab management features, the extension may access the following information through Chrome extension APIs:

  • Open tab URLs
  • Open tab titles
  • Tab window information
  • Chrome tab group state
  • Locally stored extension preferences and workspace state

If you choose to use history-based search or recovery features, the extension may request Chrome's optional history permission. This permission is used only after you grant it and only to show relevant browser history results inside the extension.

How the extension uses tab and group data

The extension uses tab and group data only to:

  • Show your open tabs in the side panel
  • Group related tabs into Chrome tab groups
  • Let you rename, collapse, expand, ungroup, hide, sync, or manage groups
  • Remember useful group/workspace state locally
  • Respect manual grouping preferences and overrides
  • Help you find relevant open tabs or, if permitted, recent history results

Data stored locally

Unmessy Tabs stores some information locally in Chrome storage on your device. This may include:

  • Custom group names
  • Manual grouping preferences and overrides
  • Remembered group/workspace state
  • Side panel UI state
  • Sort or display preferences
  • A randomly generated analytics client identifier

Locally stored tab/workspace data remains in your browser unless you clear extension data, reset Chrome storage, or uninstall the extension.

Analytics data

Unmessy Tabs uses Google Analytics 4 to collect limited product usage analytics. Analytics events may include actions such as:

  • Opening the side panel
  • Opening a new tab from the extension
  • Using auto-grouping
  • Using tab actions such as go to tab, move tab, or close tab
  • Using search
  • Changing sort mode
  • Using group actions such as rename, expand, collapse, ungroup, or delete

Analytics events may include limited non-content parameters such as action source, sort mode, group action type, and extension version.

Unmessy Tabs does not intentionally send tab URLs, tab titles, page contents, browsing history entries, search query text, group names, or personally identifying information to Google Analytics.

Floating launcher

Unmessy Tabs may show a small floating launcher on webpages so you can open the extension side panel quickly. This launcher is used only to open or close the extension interface. It does not read page contents, form fields, passwords, messages, or other website content.

Data sharing

Unmessy Tabs does not sell user data. The extension does not share tab URLs, tab titles, browsing history, page contents, group names, or locally stored workspace data with the developer or third parties.

Limited product usage analytics are shared with Google Analytics 4, a service provided by Google, for the purpose of understanding extension usage and improving the product.

Remote servers

The extension sends analytics events to Google Analytics over HTTPS. Unmessy Tabs does not operate its own backend server for storing your tab, group, browsing history, or workspace data.

Data retention

Data stored locally by the extension remains on your device until you clear extension data, reset Chrome storage, or uninstall the extension.

Analytics data sent to Google Analytics is retained according to the analytics settings configured for the Google Analytics property.

Children's privacy

Unmessy Tabs is not directed to children and does not knowingly collect personal information from children.

Chrome Web Store Limited Use

The use of information received from Chrome extension APIs will adhere to the Chrome Web Store User Data Policy, including the Limited Use requirements.

Changes to this policy

This privacy policy may be updated when the extension changes. The latest version will be available at this page.
